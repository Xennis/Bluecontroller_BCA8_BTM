## Hardware

* Crumb168 V2.3 AVR ATmega Modul
* Atmel AVR ATmega328

## Software

* Atmel Studio 6
* PuTTY

## Code Versions

### Basic

Simple code (its sends only "Hallo", "Hallo", etc.)

Setting for Putty:
* Baud: 115200
* Data bits:	8
* Stop bits:	2
* Parity: None
* Flow control: None
